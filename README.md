### الخطوة 1: أنشئ حساب على Vercel
1. اذهب إلى **[vercel.com](https://vercel.com)**
2. اضغط **Sign Up** (مجاني)
3. سجّل الدخول بحساب GitHub أو البريد الإلكتروني

### الخطوة 2: ارفع المشروع على GitHub
1. أنشئ حساب على **[github.com](https://github.com)** إذا لم يكن لديك
2. أنشئ **New Repository** جديد باسم `stars-of-the-field`
3. اتبع الخطوات لرفع ملفات المشروع:

```bash
git init
git add .
git commit -m "نجوم الملعب - المتجر الرياضي"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/stars-of-the-field.git
git push -u origin main
```

1. في لوحة تحكم Vercel اضغط **Add New... → Project**
2. اختر **Import** من GitHub لمشروع `stars-of-the-field`
3. اضغط **Deploy** ✅
